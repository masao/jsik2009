\begin{enumerate}
 \item 日時：2009年5月16日（土） 
 \item 会場：東京工業大学 大岡山キャンパス 西9号館
       （東京都目黒区大岡山２丁目１２-１）\\
       \url{http://www.titech.ac.jp/access-and-campusmap/j/o-okayamaO-j.html}\\
       （最寄り駅: 大岡山駅より徒歩1分）
 \item 参加費：無料
 \item 資料代：会員無料、一般非会員3,000円、学生非会員1,000円
 \item 情報交流会参加費：1,000円（学生無料）
 \item お問い合わせ先：年次大会実行委員会
       \url{jsik2009@valdes.titech.ac.jp}
\end{enumerate}
